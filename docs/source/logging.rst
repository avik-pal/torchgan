torchgan.logging
================

.. currentmodule:: torchgan.logging

This subpackage provides strong visualization capabilities using 
`tensorboardX <https://github.com/lanpa/tensorboardX>`_. It is strongly integrated with the
Trainer. The `Logger` supports a variety of configurations and customizations.
